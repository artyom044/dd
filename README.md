# Проект: база данных "Whoosh - система аренды самокатов"

## О проекте

Этот проект представляет собой реляционную базу данных, разработанную для системы аренды электросамокатов Whoosh.  
Система управляет:

- Пользователями и их профилями
- Парком самокатов и их состоянием
- Тарифами на аренду
- Промоакциями и скидками
- Историями поездок и оплат

База данных может служить основой для реального сервиса аренды самокатов или использоваться в учебных целях для изучения проектирования реляционных моделей и SQL-запросов.

## Сущности (таблицы)

### Основные таблицы:
- **Useer** (Пользователи): ID, имя, фамилия, телефон, email, дата регистрации, активный промокод
- **Scooter** (Самокаты): ID, модель, уровень заряда, статус, координаты, дата последнего ТО
- **Tariff** (Тарифы): ID, название, стоимость за минуту, стартовый сбор, активность
- **Promocode** (Промокоды): ID, код, процент скидки, дата истечения, максимальное количество использований

### Транзакционные таблицы:
- **Rental** (Прокаты): ID проката, пользователь, самокат, тариф, промокод, время начала/окончания, стоимость, статус
- **Payment** (Платежи): ID платежа, прокат, сумма, способ оплаты, дата оплаты
- **PromoUsage** (Использование промокодов): ID использования, пользователь, промокод, дата использования, связанный прокат

### Связи между сущностями
Схема связей построена на принципах реляционной целостности и включает следующие зависимости:
- **Useer → Rental** Один пользователь может совершить несколько прокатов (`Rental.user_id → Useer.user_id`)
- **Scooter → Rental** Один самокат может использоваться в разных прокатах (`Rental.scooter_id → Scooter.scooter_id`)
- **Tariff → Rental** Один тариф применяется в разных прокатах (`Rental.tariff_id → Tariff.tariff_id`)
- **Promocode → Rental** Один промокод может использоваться в разных прокатах (`Rental.promo_id → Promocode.promo_id`)
- **Rental → Payment** Каждый прокат имеет один платёж (`Payment.rental_id → Rental.rental_id`)
- **Useer ↔ Promocode** Многие ко многим: пользователь может использовать много промокодов, промокод может быть использован многими пользователями (через таблицу `PromoUsage`)
